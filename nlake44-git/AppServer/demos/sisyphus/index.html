Hello, admin!

            
This is Sisyphus, a log viewer for components within AppScale. See the left sidebar for logs that Sisyphus has collected, or below for general information about Sisyphus-aware components within AppScale. Dimmed out components do not yet support Sisyphus, but could benefit from it!

AppController

              
The AppController is a daemon that automatically configures and deploys all the other AppScale components. For components that run across multiple machines, a single AppController starts each machine's component in the correct order, as needed by the application. 

            

            

              
AppLoadBalancer

              
The AppLoadBalancer serves two functions: (1) It routes users to their applications, with the assistance of nginx and haproxy, and (2) provides a web interface that users can log in via and administrators can examine the status of their AppScale deployment. 

            

            

              
AppServer

              
The AppServer is a version of the open source Google App Engine SDKs that has been modified to provide scalability via open source software or other AppScale components. 

            

          

          

            

              
PBServer

              
The Protocol Buffer Server (PBServer) provides AppServers with a database-agnostic way to save and retrieve data. It pulls in library support for any database that AppScale runs over to provide a single API for data access. 

            

            

              
UAServer

              
The UserAppServer (UAServer) exposes a SOAP interface that allows callers to query data related to users or applications, and modify that data as needed. New users and apps can be created, and old ones can be deleted. 

            

            

              
Loki

              
Loki provides fault-tolerance testing mechanisms within AppScale. It is off by default, but if turned on, it will randomly kill components within AppScale, and see if they come back up like they should.
